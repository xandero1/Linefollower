## specifications
<br />
microcontroller: Arduino nano
<br />
motors: Gearmotors 9V 50:1
<br />
h-bridge: DRV8833
<br />
sensors: QTR-8A
<br />
batteries: 2X SamsungNR18650
<br />
wireless communication: HC-05
<br />
distance sensor - motors: 125mm
<br />
weight: 386g
<br />
speed: Te bepalen
